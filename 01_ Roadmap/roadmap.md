# 6–12 Month Roadmap

## Step 1: Strengthen Core IT + Security Basics
- Networking refresh: TCP/IP, DNS, VPNs, firewalls (Cisco Packet Tracer, free labs)
- OS administration: Windows Server (AD, Group Policy), Linux basics (permissions, SSH, patching)
- Security fundamentals: Google Cybersecurity Professional Certificate (Coursera/ Google Skills), CompTIA Security+ Preparation (free resources)

👉 Outcome: Ready for System Admin / Network Admin roles

---

## Step 2: Cloud & Automation
- Cloud skills: AWS/Azure basics (IAM, security groups, monitoring)
- Automation: PowerShell + Python scripting
- Projects: automate user creation, log parsing, patch deployment

👉 Outcome: Target Cloud Support Engineer / Infrastructure Engineer roles

---

## Step 3: Hands-On Security Practice
- Labs: TryHackMe (SOC paths), HackTheBox beginner labs
- SIEM tools: Splunk Free, Microsoft Sentinel, Elastic
- Document labs/projects on GitHub or LinkedIn

👉 Outcome: Demonstrable security skills for resume

---

## Step 4: Transition Preparation
- Apply for IT roles with security responsibilities
- Network with cybersecurity professionals
- Earn Security+ certification (Vendor-neutral, entry-level but deeper cybersecurity exam) or Microsoft SC-900 certification (Fundamentals-level, focused on Microsoft ecosystem)
- Aim for SOC Analyst / Junior Security Engineer positions
